## First section heading

Body text goes here. Write in Markdown. Some tips:

- Use `##` for major sections and `###` for subsections
- Keep paragraphs short — 2-4 sentences reads best on screen
- Use code blocks with language hints for syntax highlighting:

```python
print("like this")
```

- Link to things: [text](https://example.com)
- Use *italics* for emphasis and **bold** sparingly
- Blockquotes work well for asides and analogies:

> This is a callout or aside. Use it for the thing you wish someone had told you.

## Second section heading

Continue writing. Add as many sections as you need.

If you're writing about a project, a good structure is:

1. **What** — what did you build and why
2. **How** — the interesting technical bits, what worked, what didn't
3. **What you learned** — takeaways, surprises, things you'd do differently

## Wrap up

End with a summary, next steps, or a link to the project.
