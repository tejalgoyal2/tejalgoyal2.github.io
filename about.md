Hi, I'm Tejal.

By day I work in cybersecurity. On the side, I build things — mostly because I want them to exist, partly because building is how I learn best.

This blog is where I write about projects as they happen. That includes the dead ends, the APIs that silently ignore you, and the workarounds you end up shipping instead. If something took me a while to figure out, it probably goes here.

## What you'll find

Posts about building in public: dev write-ups with code and context, notes on vibe coding with AI tools, and occasionally whatever else I feel like writing about. Expect macOS development, AI-assisted coding, and side project adventures.
